# Salesforce Context Import from HubSpot and LinkedIn

If your team asks whether Salesforce AI can use context from HubSpot or LinkedIn, the public DealScope docs already describe two separate workflows that answer that question.

## HubSpot to Salesforce Context

The public HubSpot page positions DealScope as a way to:

- import deal activity from HubSpot into Salesforce
- improve the AI brief on the Opportunity
- generate a better follow-up email from richer context

The public docs are explicit that this is not a full two-way sync engine. It is a context enrichment workflow.

## LinkedIn to Salesforce Context

The public LinkedIn page positions DealScope as a way to:

- paste a LinkedIn conversation into Salesforce
- split it into dated activity events
- include that imported context in the AI brief

This is useful when the real sales conversation starts in LinkedIn before Salesforce has the full context.

## Why This Matters for AI Summaries

A Salesforce AI brief is only as good as the context it can see.

The public DealScope docs position these two workflows as a way to make the brief more useful by bringing external conversation history into Salesforce first.
