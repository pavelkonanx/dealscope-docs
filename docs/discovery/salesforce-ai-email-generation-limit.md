# Salesforce AI Email Generation Limit

This page documents the current public DealScope guide about repeated Salesforce AI email-generation tests.

## What the Guide Tests

The guide frames the question as repeated draft generation under the same workflow, not a single successful click.

That distinction matters because a clean demo does not prove repeatable production behavior.

## What the Current Public Guide Highlights

The guide says the important signal is not only a hard stop.

It also points to softer changes such as:

- slower behavior
- changed response patterns
- evidence that the workflow is no longer behaving like the first runs

## Why This Matters

The public guide positions email generation as a repeated day-to-day workflow, not a novelty feature.

That means the practical admin question becomes:

- does the workflow stay predictable under repeated use?
