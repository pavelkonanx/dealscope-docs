# HubSpot to Salesforce Activity Import with DealScope

The public DealScope site describes a focused workflow for bringing HubSpot deal activity into Salesforce so the AI brief has better source context.

## What It Does

The public page says DealScope can use:

- HubSpot emails
- notes
- meetings
- calls
- other deal activity

to improve the Salesforce brief and follow-up workflow.

## Supported Record Type

The current public page explicitly positions this workflow on Salesforce Opportunity records.

## Why This Exists

The public page says many teams keep early deal history in HubSpot while active work happens in Salesforce.

That creates three common problems:

- activity history stays in HubSpot
- Salesforce users lose context
- the AI summary becomes weaker because it cannot see the missing history

## Public Workflow

The current public page describes this sequence:

1. Copy the HubSpot deal URL.
2. Paste the deal URL into DealScope.
3. Import relevant HubSpot activity into Salesforce.
4. Re-run or enrich the AI brief on the record.
5. Use the improved draft follow-up email.

## What It Improves

The public page says imported HubSpot context can improve:

- summary quality
- buying signals
- risk visibility
- stakeholders
- draft email quality

## Product Boundary

The public page is clear that this is not a full integration replacement.

It describes the native HubSpot and Salesforce integration as the better fit for broader system sync, while DealScope is positioned as:

- focused on the deal you care about now
- fast for reps, admins, and consultants
- useful when you need context immediately
